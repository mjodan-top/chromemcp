# Football Vocab - 足球单词学习游戏设计文档

**日期**: 2026-02-20
**目标**: 创建一个结合足球游戏的英语单词学习 H5 应用，提升中学生学习兴趣和效果。

---

## 1. 项目概述

### 1.1 核心概念
将英语单词学习与足球游戏机制结合，通过**进球奖励机制**让学习变得有趣。每次答对题目就像踢进一球，连续答对触发"连胜模式"。

### 1.2 目标用户
- 中学生（12-18岁）
- 英语词汇量：中考/托福基础水平
- 使用场景：课后复习、碎片时间学习

### 1.3 成功标准
- 游戏化降低学习枯燥感
- 三种模式适应不同学习需求
- 错题本帮助针对性复习
- 进度统计可视化学习成果

## 2. 游戏模式设计

### 2.1 训练模式（Practice Field）
**目标**: 无压力学习，熟悉单词

**机制**:
- 卡片式浏览，左右切换
- 显示单词 + 音标 + 中文释义 + 例句
- 点击喇叭图标播放发音（Web Speech API）
- 无时间限制，无分数压力

**界面元素**:
- 单词卡片（大字体显示）
- 音标和发音按钮
- 中文释义
- 例句（英文+中文）
- 左右切换箭头
- 当前进度指示器

### 2.2 射门挑战（Shooting Challenge）
**目标**: 限时答题，测试反应和记忆

**机制**:
- 90秒倒计时
- 每题显示单词 + 4个中文选项
- 答对：球员射门动画，球进网，+10分
- 答错：球被扑出/射偏，显示正确答案
- 连续答对3题触发"火力全开"（特效+双倍分数）
- 时间到后显示最终得分和评价

**评分系统**:
- 普通进球：+10分
- 火力全开进球：+20分
- 连击加成：每连击+1分额外奖励

**评价等级**:
- 0-50分：新手球员
- 51-100分：替补队员
- 101-150分：主力球员
- 151-200分：球队核心
- 200+分：世界级球星

### 2.3 传球关卡（Passing Levels）
**目标**: 连续闯关，深度记忆

**机制**:
- 共10关，难度递增
- 每关需要连续答对3-5题才能过关
- 答错一题即回到本关起点
- 每关完成后解锁新单词集合
- 最终关卡需要连续答对5题完成"绝杀进球"

**关卡设计**:
```
Level 1: 简单词 3题连续
Level 2: 简单词 4题连续
Level 3: 简单词 5题连续
Level 4: 中等词 3题连续
Level 5: 中等词 4题连续
Level 6: 中等词 5题连续
Level 7: 困难词 3题连续
Level 8: 困难词 4题连续
Level 9: 困难词 5题连续
Level 10: 混合词 5题连续（绝杀）
```

## 3. 词汇系统设计

### 3.1 内置词库（200词）

**简单级别（60词）**:
- 颜色：red, blue, green, yellow, black, white...
- 数字：one到twenty
- 动物：dog, cat, bird, fish, tiger...
- 水果：apple, banana, orange, grape...
- 家庭成员：father, mother, brother, sister...

**中等级别（80词）**:
- 学校：classroom, textbook, homework, examination...
- 食物：breakfast, vegetable, restaurant, delicious...
- 日常：morning, afternoon, evening, weather...
- 运动：basketball, swimming, running, exercise...

**困难级别（60词）**:
- 科学：experiment, electricity, chemical, biology...
- 科技：computer, internet, program, digital...
- 抽象：knowledge, experience, culture, society...

### 3.2 数据结构

```typescript
interface Word {
  id: string;
  word: string;
  phonetic: string;
  meaning: string;
  example: string;
  exampleTranslation: string;
  difficulty: 'easy' | 'medium' | 'hard';
  category: string;
}

interface UserProgress {
  learnedWords: string[];
  wrongWords: string[];
  highScore: number;
  maxStreak: number;
  customWords: Word[];
  levelProgress: number; // 闯关进度
}
```

### 3.3 自定义单词

**添加流程**:
1. 点击"添加单词"按钮
2. 输入英文单词（自动获取音标）
3. 输入中文释义
4. 输入例句（可选）
5. 保存到 LocalStorage

**自定义单词特性**:
- 可编辑和删除
- 标记为自定义标签
- 混入游戏题库参与游戏

## 6. 交互流程

### 6.1 应用启动流程

```
页面加载
  ↓
检查 LocalStorage 是否有用户数据
  ↓
加载内置词库 + 自定义词库
  ↓
显示主界面（默认训练模式）
  ↓
显示第一个单词
```

### 6.2 游戏流程

**射门模式**:
```
点击"射门模式"
  ↓
显示倒计时 3-2-1
  ↓
加载第一道题目
  ↓
玩家选择答案
  ↓
判断正误 → 播放动画 → 更新分数
  ↓
时间到或玩家退出
  ↓
显示成绩和评价
```

**闯关模式**:
```
点击"闯关模式"
  ↓
显示当前关卡信息
  ↓
开始答题（连续3-5题）
  ↓
答错 → 显示失败 → 回到本关起点
  ↓
答对所有题 → 过关动画 → 解锁下一关
  ↓
全部10关通过 → 显示通关画面
```

### 6.3 用户操作

| 操作 | 响应 |
|------|------|
| 点击模式标签 | 切换游戏模式 |
| 点击选项按钮 | 提交答案，播放反馈动画 |
| 点击喇叭图标 | 播放单词发音 |
| 点击"添加单词" | 打开添加表单 |
| 点击"错题本" | 显示错题列表 |
| 点击"统计" | 显示学习数据图表 |

## 7. 功能详细设计

### 7.1 单词发音

**实现方式**:
- 使用 Web Speech API
- 点击喇叭图标触发
- 语音：美式英语（en-US）
- 语速：0.9（稍慢，便于学习）

**降级方案**:
- 如果浏览器不支持，隐藏喇叭图标
- 显示提示："您的浏览器不支持发音功能"

### 7.2 错题本

**功能**:
- 自动记录答错的单词
- 显示错题次数
- 可进入"错题复习"模式
- 答对3次后从错题本移除

**界面**:
- 列表展示：单词 + 答错次数 + 最后答错时间
- 复习按钮：进入专门的错题练习模式
- 清空按钮：确认后清空所有错题

### 7.3 学习统计

**统计数据**:
- 总学习单词数
- 总体正确率
- 最高连胜记录
- 射门模式最高分
- 闯关模式进度
- 各难度单词掌握情况

**展示方式**:
- 数字卡片展示关键数据
- 简单的进度条/柱状图
- 成就徽章系统

### 7.4 数据持久化

**LocalStorage 键值**:
```javascript
{
  'football_vocab_progress': {
    learnedWords: ['word1', 'word2', ...],
    wrongWords: [{wordId: 'xxx', count: 3}, ...],
    highScore: 150,
    maxStreak: 8,
    levelProgress: 5,
    totalGames: 20
  },
  'football_vocab_custom': [
    {id: 'custom1', word: '...', ...},
    ...
  ]
}
```
